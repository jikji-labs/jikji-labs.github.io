About Jikji Labs

Jikji Labs builds Jikji — a modular agent platform set in metal type. We think agent infrastructure should be composable, governed, and durable from day one, the way movable type made publishing reliable at scale.

Jikji is a single static Go binary that runs a deterministic agent loop, adaptive multi-model orchestration, a learning memory plane, a native ontology engine, and enterprise governance — from a laptop to a multi-node cluster.

The project is under active development and the source is not yet public; it will be open-sourced soon. If you are interested in the platform, an early look, or working together, we would love to hear from you.
