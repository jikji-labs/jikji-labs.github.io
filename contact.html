About Jikji Labs

Jikji Labs builds Jikji — a modular agent platform set in metal type. We think agent infrastructure should be composable, governed, and durable from day one, the way movable type made publishing reliable at scale.

This page remains authoritative in English. The selected locale applies to shared navigation; do not rely on an unofficial translation for legal terms.

Jikji is a modular Go runtime for deterministic agent loops, adaptive multi-model orchestration, a learning memory plane, a native ontology engine, and enterprise governance — deployable from a laptop to a multi-node cluster.

The public source distribution is being prepared under GPL-3.0-only by default. Organizations may request a separate written Apache-2.0 grant for a specified release. Jikji remains pre-1.0, so early adopters should plan upgrades around published release evidence and known limitations.
