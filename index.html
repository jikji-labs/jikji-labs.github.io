The Name

Named after the world's first movable metal type book

Jikji (직지심체요절) was printed in Korea in 1377 — the oldest surviving book printed with movable metal type, decades before Gutenberg. Movable type changed publishing the same way Jikji aims to change agents: composable pieces, set precisely, producing reliable output at scale.

Every module carries a letterpress name — the foundry casts the runtime, type faces the models, sorts are the tools, the compositor sets sessions, the forme locks the agent loop, and the press serves it to the world. Even the default port is 1377.

Why Jikji

What makes it different from other agent frameworks

Most agent frameworks are Python libraries glued to a vector DB, with opaque loops and governance bolted on later. Jikji is built like infrastructure from day one.

01

One static binary, zero dependency hell

Written in Go and shipped as a single static binary. No Python environments, no dependency conflicts. The same binary runs on your laptop, as split services, or in Kubernetes with the included Helm chart.

02

A deterministic, recoverable agent loop

The forme loop models every step as explicit actions and observations — with checkpointing and recovery built into the engine. Agent runs are replayable and auditable, not opaque chains that vanish on crash.

03

Governance built in, not bolted on

Multi-tenancy, scoped API keys, RBAC, structured audit trails with retention, policy gates on tool use, and human-in-the-loop approvals — all in the core, plus PII/secret redaction with locale-aware patterns.

04

Scale-free: laptop → services → grid

Local-first durable state on SQLite. Start in one process, split into services, then push tool execution to remote nodes over TLS relay tunnels — without changing your agents.

05

Memory that actually learns

The plate cognitive memory plane extracts facts from conversations, consolidates them in background "dreaming" passes, and serves hybrid recall with context compaction. lexicon builds a knowledge graph from documents with contradiction detection.

06

Open protocols, no lock-in

Serves an OpenAI-compatible API and speaks MCP (server and client), ACP, and A2A for agent-to-agent collaboration. Works with OpenAI-compatible, Anthropic-compatible, Ollama, and GLM/Z.AI backends.

Architecture

A print shop of composable modules

Each module owns one concern behind a strict boundary. Together they compose into a full agent platform.

forme 판

Deterministic agent loop — delegation, HITL, Lua hooks, checkpoints, step events.

galley 게라

Shared durable SQLite records and links for multi-agent collaboration.

dispatch 배차

Tenant/provider/model scheduling, bounded queues, retry with backoff.

press 인쇄기

HTTP ingress — OpenAI-compatible API, SSE streaming, idempotency.

type 활자

Inference provider abstractions across OpenAI/Anthropic/Ollama-compatible backends.

sorts 낱자

Tool execution, MCP integration, and runner abstractions.

compositor 식자공

Session orchestration, queues, and flow composition.

plate 연판

Cognitive memory — fact extraction, dreaming, hybrid recall, compaction.

lexicon 사전

Knowledge graph extraction from documents with conflict resolution.

imprimatur 출판인가

Tenancy, API-key auth, scopes, RBAC, and audit primitives.

almanac 책력

Deterministic scheduling — cron, interval, batch, maintenance work.

foundry 주조소

Bootstrap, configuration, doctor checks, orchestration control.

Interoperability

Speaks every protocol that matters

OpenAI-compatible API

Drop-in /v1/chat/completions, models, runs, and sessions — point existing clients at Jikji.

MCP

Model Context Protocol server and client over stdio — consume MCP tools or expose Jikji as one.

ACP

Agent Communication Protocol over stdio for local process-based agent peers.

A2A

Agent-to-agent collaboration — handoff, delegation, and result harvesting between agents.

Operator tooling

jikjictl covers credentials and vault, RBAC, policy, audit queries, tracing, usage, and a marketplace — plus a separate web dashboard that never stores keys server-side.

Terminal-native

Interactive TUI chat, single-prompt mode, JSONL event streams for automation, and an MCP-shaped JSON-RPC mode — agents you can pipe.

Tested like infrastructure

A deterministic make verify gate plus guarded live smoke tests against real providers. Benchmark, fuzz, property, and concurrency tests throughout the codebase.

Get Started

Running in under a minute

# start the server (default port 1377 — yes, that 1377)
go run ./cmd/jikji serve --config config.example.yaml

# check your local environment
go run ./cmd/jikji doctor

# open the interactive chat TUI
go run ./cmd/jikji chat --mode tui

# or call the OpenAI-compatible API
curl -X POST http://127.0.0.1:1377/v1/chat/completions \
-H 'Content-Type: application/json' \
-d '{"model":"jikji-echo","messages":[{"role":"user","content":"hello"}]}'
